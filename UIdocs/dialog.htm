代码如下:
var mm = new sl.ui.dialog(document.body, { message: "内容", title: "头部", footer: "底部", zIndex: 10002 });
$("#btnOpen").click(function () {
        mm.open();
 });
效果:
名称 说明
title{string} 标题
footer{string} 底部
message{string} 中间的消息体
zIndex{number} 默认为1000
showClose{boolean} 是否显示关闭按钮 默认为true
autoShow{boolean} 一开始就出现 默认为false
centerX{boolean} 对话框水平居中 默认为true
centerY{boolean} 对话框垂直居中 默认为true
showOverlay{boolean} 是否显示遮罩层 默认为true
overlayCSS{object} 遮罩层的样式
