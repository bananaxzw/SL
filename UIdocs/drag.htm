拖动


代码如下:
new sl.ui.draggable(document.getElementById("Div2"), { proxy: "clone" });
效果:
无限制拖拉
代码如下:
new sl.ui.draggable(document.getElementById("sfs"), { proxy: "", containment: document.getElementById("Div1") });
效果:
内部拖拉

名称 说明
proxy 拖拉时候是否采用影子拖拉 目前只提供"clone"和""(采用源对象拖拉)
handle 拖拉的手柄 也就是鼠标移动到元素的指定区域才能拖拉
containment 拖拉的容器 为dom元素
