拖放


代码如下:
            new sl.ui.draggable(document.getElementById("d1"), { proxy: "clone", revert: true });
            new sl.ui.draggable(document.getElementById("d2"), { proxy: "clone" });
            new sl.ui.draggable(document.getElementById("d3"), { proxy: "clone" });
            new sl.ui.droppable(document.getElementById("dd"), {
                accept: '#d1,#d3',
                onDragEnter: function (e, source) {
                    slChain(this).html('enter' + source.id)
                },
                onDragLeave: function (e, source) {
                    slChain(this).html('leave' + source.id)
                },
                onDrop: function (e, source) {
                    slChain(this).html(slChain(source).html() + ' dropped');
                }
            });
效果:
拖拉
Apple
Peach
Orange
停放

名称 说明
accept 接受拖拉元素的选择器
onDragEnter 进入触发的事件
onDragLeave 离开触发的事件
onDrop 鼠标松开的时候
