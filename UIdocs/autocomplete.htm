自动补全autocomplete


代码如下:
var ss = new sl.ui.autocomplete(document.getElementById("basic"), 
{
 min: 1, 
 source: ["许志伟1", "许志伟2", "许志伟3", "许志伟4", "许志伟5"] 
});
效果:
输入"许"看看 可上下键操作
 
代码如下:
  var tt = new sl.ui.autocomplete(document.getElementById("Text1"), { min: 1,
                source:
                [{ "text": "许志伟1", "value": "沙发上" }, { "text": "许志伟2", "value": "3" }, { "text": "许志伟3", "value": "3"}],
                selected: function (event, ui) {
                    alert(ui.text);
                    alert(ui.value);
                }
});
效果:
输入"许"看看 可上下键操作
 
代码如下:
 var mm = new sl.ui.autocomplete(document.getElementById("ajax"),
           {
               min: 1,
               dynamicSource: true,
               ajaxOption: { url: "../ajaxdata.ashx",
               extendData: { name: "xuzhiwei", age: 11 }
            }
});
效果:
输入"许"看看 可上下键操作
 
名称 说明
min 产生自动补全的字符数 默认为1
source 自动补全的数据源 如果设置远程数据源的话 可不设置
selected 选中下拉列表项的事件 第一个参数为事件对象
第二个参数为项的数据对象ui 包括ui.text ui.value
dynamicSource 如果要设置远程数据源 请把设置为true
ajaxOption 远程数据请求选项
例如: { url: "", extendData: {} }
url表示请求数据源地址
extendData请求数据源地址的参数
