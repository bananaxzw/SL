类概要

array扩展

方法概要

名称 说明
addRange(arr, items) 批量添加元素
clear(arr) 清空数据
contains(arr, obj) 数组时候存在某元素
deleteRepeater(arr) 移除重复
every(arr, f, oThis) 验证所有元素是不是符合你的规则
filter(arr, f, oThis) 按照指定规则过滤数据 并返回新数据
forEach(arr, f, oThis) 循环遍历数据 并指定函数动作
indexOf(arr, obj) 查找元素在数组中的位置，如果不存在则返回-1
insertAfter(arr, obj, index, isReturnNew) 在指定位置后插入元素
insertBefore(arr, obj, Index, isReturnNew) 在指定位置前插入元素
lastIndexOf(arr, obj) 查找元素在数组中的位置，如果不存在则返回-1
map(arr, f, oThis) 按照指定规则把数组映射到新的数组中
remove(arr, item) 移除指定元素
removeAt(arr, Index) 移除指定位置处的元素
replace(arr, obj, Index, isReturnNew) 替换指定位置处的元素
some(arr, f, oThis) 验证有存在符合你的规则的元素
